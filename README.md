# My Bootstrap Portfolio

A simple and clean portfolio website built using Bootstrap 5.

## Features
- Responsive design (works on mobile, tablet, and desktop)
- Navigation bar with smooth scrolling
- About me section with education info
- Skills section with categorized badges
- Projects showcase
- Contact form and social links

## Technologies Used
- HTML5
- CSS3
- Bootstrap 5
- Bootstrap Icons
- JavaScript (for basic interactions)

## How to Use
1. Download all files
2. Open `index.html` in your browser
3. Customize the content in the HTML file

## Customization
- Change the name and profile image
- Update skills and projects
- Modify colors in the style section
- Add your contact information
